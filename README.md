## 📖 About The Project

**Paulify** is a fully responsive fashion e-commerce frontend that simulates a real-world online clothing store. It features a dynamic product carousel, category browsing, product detail pages, a functional cart system, and a clean UI — all without any frameworks or backend dependencies.

### 🎯 Why Paulify?

- 🚫 No frameworks — pure vanilla web technologies
- 📱 Mobile-first responsive design
- ⚡ Lightweight and fast-loading
- 🎨 Clean, modern UI/UX with attention to detail
- 🛒 Real e-commerce flow: Browse → View → Add to Cart

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

### Installation

```bash
# Clone the repository
git clone https://github.com/rspaul11/Paulify.git

# Navigate to project directory
cd Paulify

# Open in browser (macOS)
open index.html

# Open in browser (Linux)
xdg-open index.html

# Open in browser (Windows)
start index.html
```

Or simply download the ZIP and open `index.html` in your browser.
