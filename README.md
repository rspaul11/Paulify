## ✨ Features

- **Hero Carousel** – Eye-catching promotional slider with smooth transitions
- **Smart Search** – Real-time product search in the navbar
- **Category Navigation** – Beautiful visual category cards
- **Shopping Cart** – Fully functional cart with real-time counter
- **Product Details** – Dedicated pages with rich information
- **Responsive Design** – Flawless experience across all devices
- **Newsletter Signup** – Clean subscription section
- **Multi-page Layout** – Home, Shop, Categories, About & Contact

## 🚀 Getting Started

### Quick Start

```bash
# Clone the repository
git clone https://github.com/rspaul11/Paulify.git

# Go to project folder
cd Paulify

# Open in your browser
open index.html
```

Or simply download the project and open `index.html`.
